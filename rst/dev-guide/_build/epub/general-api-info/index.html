General API Information

The Cloud Keep API is defined as a RESTful HTTP service that uses all aspects of the HTTP protocol, including methods, URIs, media types, and response codes. Review the topics in this section to learn more about these API components and how to access and use the API for this service.

The Cloud Keep API supports JSON data serialization request and response formats.

Note

All requests to authenticate against and operate the service are performed using SSL over HTTP (HTTPS) on TCP port 443.
